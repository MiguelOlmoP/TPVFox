# Gestion tienda - tpvfox
## Introduccion
Una aplicacion para gestionar una pequeña y mediana empresa. Donde pueda cobrar tickets, realizar factura y la gestion compra.
## Lenguajes de programción utilizados
Principalmente esta basada en php,html,css y javascript.
Pero tambien utilizamos:
1. Bootstrap (framework)
2. Python -> Para modulo importar de SPPG
## Objetivo principal.
* Un puesto (TpvPos) sencillo y rapido, con teclado ... no esta pensado para pantalla tatil
* Poder importar de forma sencillo los datos que tengamos en:
  - SPPG TPVElite (Sage)
  - Virtuemart
